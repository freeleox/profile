# 2026-03-25 工作日志

## TapTap预约榜爬虫脚本
- 创建了 `taptap-crawler.user.js` 篡改猴脚本
- 功能：爬取 TapTap 预约榜（https://www.taptap.cn/top/reserve）游戏数据
- 一级数据：游戏名、设备支持、备注
- 二级数据：预约榜排名、预约人数、上线日期、标签、厂商
- 支持自动翻页、进度条显示、Excel导出（使用 SheetJS/xlsx 库）
- 左下角按钮触发，弹窗输入爬取数量

## Bug修复：预约榜排名重复
- 问题：从二级页面爬取排名时，多个游戏返回相同排名
- 原因：二级页面的 `#数字` 选择器匹配不准确
- 修复：改为从一级页面的 `<span class="rank-index">` 元素提取排名数字，更准确可靠
- 移除了二级页面中的排名爬取逻辑
